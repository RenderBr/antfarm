# Neural Ant Farm 🐜🧠

An open-world 2D ant colony where **every ant is its own neural network**. Nothing is
scripted — foraging, eating, digging and building all **emerge** and **evolve** as the
queen breeds the most successful ants over generations.

## Run

Requires [Bun](https://bun.sh). Install deps, then build & serve:

```bash
bun install
bun run dev        # build once, then serve at http://localhost:5173
```

Edit TypeScript in `src/` or styles in `src/input.css`, then `bun run build`
(or `bun run watch` to rebuild on change) and refresh the browser.

## What's actually simulated

**Each ant = one brain.** A small feed-forward neural net:
- **22 sensory inputs** — three forward probes (blocked? food? food-trail? home-trail?),
  plus energy, what it's carrying, heading-to-nest (sin/cos), distance home, food underfoot,
  an internal oscillator, and bias.
- **9 action outputs** — turn, move, dig, build, grab, drop, lay food-trail, lay home-trail, eat.

**Open, mutable world** (tile grid):
- Ground, diggable **dirt**, indestructible **rock**, the **nest** core, and **walls** the
  ants build themselves.
- Ants **dig** dirt into carried soil and **build** it back as walls anywhere — they reshape
  their world.
- Two evaporating/diffusing **pheromone fields** (food trail + home trail) ants can read and lay.
- Food clusters that the world keeps replenishing.

**Survival & evolution (genetic algorithm, no backprop):**
- Ants spend energy moving/working and must **eat field food** to survive.
- Delivering food to the nest builds the **colony store**, which the **queen spends on eggs**.
- New ants are bred from a **gene pool of the most successful past ants** via tournament
  selection + crossover + mutation. Watch **average fitness** climb as the colony learns.

## Controls

- **Scroll** to zoom, **drag** to pan.
- **Inspect tool**: click any ant to read its live brain outputs, senses, energy and stats.
- **Food / Dirt / Wall / Erase** tools (with brush size) to sculpt the world yourself.
- Speed slider, pause, new world, and toggles for pheromone view and an ant's vision rays.

## Files

- `src/nn.ts` — evolvable neural network + genetic operators
- `src/world.ts` — tile world, food, pheromone fields, generation, dig/build
- `src/ant.ts` — per-ant sensing and actions driven by its brain
- `src/spider.ts` — predator (hunts ants; also evolves)
- `src/worm.ts` — burrowing organism that enriches soil
- `src/sim.ts` — colony, queen, gene pool, evolution loop
- `src/app.ts` — rendering (camera + layers), tools, UI, main loop
- `src/input.css` — Tailwind v4 entry (theme + component + JS-markup styles)

Build output is emitted to `dist/` (regenerated by `bun run build`). `serve.js` is a tiny
Bun static server used by `bun run dev` / `bun run serve`.
